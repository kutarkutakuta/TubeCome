プライバシーポリシー

最終更新: 2026-01-10

1. 本ポリシーについて

TubeCome （以下「本サービス」）は、利用者のプライバシーを尊重し、個人情報の保護に努めています。

2. 収集・保存する情報

  • ローカル保存：チャンネル登録、お気に入り、テーマ設定、動画リスト、コメントといったこれらの情報はブラウザ内に保存されます。
  • Cookie：サービスの改善、利用状況の分析および内部 API 用にCookieを使用します（セキュア属性で設定し、ブラウザから直接参照不可）。
  • API 利用ログ：YouTube Data API リクエストの種別と使用量を集計する目的で利用します（個人を特定する目的では利用しません）。
  • アクセス情報：IP アドレス等をサーバーアクセスログで一時的に保持する場合があります（不正利用防止・障害解析目的）。

3. 情報の利用目的

  • サービス提供・維持・改善のため
  • 利用状況の把握と不正利用の防止のため
  • API 使用量の制御と安定運用のため

4. 外部サービス

  • YouTube RSS / YouTube Data API など、コメント取得・チャンネル解決のために外部サービスへリクエストを送信します。
  • 第三者トラッキング（広告・解析ツール）は使用していません。

5. データの保持と削除

ブラウザの「サイトデータ削除」を実行すると、ローカルに保存されたチャンネル・お気に入り・テーマ設定・動画リスト・コメント・一時データ、Cookie も含めて削除されます。

6. 免責事項

本サービスは公式 YouTube とは一切関係ありません。本サービスの利用によって生じたいかなる損害についても、運営者は責任を負いかねます。

7. ポリシーの変更

必要に応じて本ポリシーを更新する場合があります。変更後の内容は本ページに掲載した時点から効力を持ちます。
